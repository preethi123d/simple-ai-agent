# 🤖 Simple AI Agent (No API)

## 📌 Overview
This project demonstrates a simple agentic workflow without using any external APIs.

## 🧠 Features
- Planner (detects user intent)
- Tool selection mechanism
- Dynamic execution logic
- Basic memory system

## ⚙️ How it Works
1. User input is analyzed
2. Intent is detected
3. Appropriate tool is selected
4. Response is generated dynamically

## 💡 Example
Input:
"I have exam in 5 days"

Output:
Dynamic study plan based on number of days

## 🚀 Tech Used
- Python
- Rule-based logic
